Markdown

Convey supports markdown, specifically a subset of CommonMark, so authors can describe the structure of their contributions and thus influence how they are displayed. This page gives a brief overview of the options available, for more information please see the CommonMark Specification.

Thematic Breaks

Thematic Breaks are formed by three matching -, _, or * symbols.

Markdown
Result
abc
---
def

abc

def
abc
___
def

abc

def
abc
***
def

abc

def

Headings

Headings are formed by up to 6 # symbols, followed by a space, followed by a string of characters.

Markdown
Result
# Heading 1

Heading 1
## Heading 2

Heading 2
### Heading 3

Heading 3
#### Heading 4

Heading 4
##### Heading 5
Heading 5
###### Heading 6
Heading 6

Lists

Ordered

Ordered lists are formed by a starting number, followed by either a . or ) symbol, followed by a space, and finally followed by the first item. Subsequent items are added after a newline, and also start with a number, a matching . or ), a space and a string of characters. New lists can be started by switching from . to ) or visa versa.

Markdown
Result
1. Item 1
2. Item 2
3. Item 3
  1. Item 1
  2. Item 2
  3. Item 3
1) Item 1
2) Item 2
3) Item 3
  1. Item 1
  2. Item 2
  3. Item 3
10) Count From 10
10) And Don't Worry
10) Numbers Auto-Increment
  1. Count From 10
  2. And Don't Worry
  3. Numbers Auto-Increment
1) Item A1
2) Item A2
1. Item B1
2. Item B2
  1. Item A1
  2. Item A2
  1. Item B1
  2. Item B2

Unordered

Unordered or bulleted lists are formed by either a -, +, or * symbol, followed by a space, and finally followed by the first item. Subsequent items are added after a newline, and start with a matching -, +, or *, a space and a string of characters. New lists can be started by alternating between -, +, and *.

Markdown
Result
- Item 1
- Item 2
- Item 3
  • Item 1
  • Item 2
  • Item 3
+ Item 1
+ Item 2
+ Item 3
  • Item 1
  • Item 2
  • Item 3
* Item 1
* Item 2
* Item 3
  • Item 1
  • Item 2
  • Item 3
- Item A1
- Item A2
+ Item B1
+ Item B2
* Item C1
* Item C2
  • Item A1
  • Item A2
  • Item B1
  • Item B2
  • Item C1
  • Item C2

Line Breaks

Soft Line Breaks

Sentences separated by a single newline are combined into one paragraph.

Markdown
Result
Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Nulla a scelerisque neque.
Cras viverra dui a tellus vestibulum, nec semper neque tincidunt.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla a scelerisque neque. Cras viverra dui a tellus vestibulum, nec semper neque tincidunt.

Hard Line Breaks

Hard line breaks are used to separate two sentences with a newline. They are formed by either a \ symbol or two spaces.

Markdown
Result
Lorem ipsum dolor sit amet, consectetur adipiscing elit.\
Nulla a scelerisque neque.\
Cras viverra dui a tellus vestibulum, nec semper neque tincidunt.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Nulla a scelerisque neque.
Cras viverra dui a tellus vestibulum, nec semper neque tincidunt.
Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Nulla a scelerisque neque.
Cras viverra dui a tellus vestibulum, nec semper neque tincidunt.

Nulla a scelerisque neque.
Cras viverra dui a tellus vestibulum, nec semper neque tincidunt.

Emphasis

Emphasis can be placed on text by surrounding it with * or _ symbols.

Markdown
Result
Lorem *ipsum* dolor sit amet, consectetur adipiscing elit.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Lorem _ipsum_ dolor sit amet, consectetur adipiscing elit.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.

Strong Emphasis

Strong emphasis can be placed on text by surrounding it with two * or _ symbols.

Markdown
Result
Lorem **ipsum** dolor sit amet, consectetur adipiscing elit.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Lorem __ipsum__ dolor sit amet, consectetur adipiscing elit.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.

Links

Links consist of three parts; the text that will be displayed, the destination URI of the link, and an optional title displayed in a tooltip. Links are formed of a pair of [] symbols enclosing the text, and a pair of () symbols enclosing the destination and optional title.

Markdown
Result
[Aletheia Ware](https://aletheiaware.com) Aletheia Ware
[Aletheia Ware](https://aletheiaware.com "Aletheia Ware Website") Aletheia Ware
[Contact Support](support@aletheiaware.com) Contact Support

Autolinks

Simple links without a text and title can be automatically created by enclosing the URI in a pair of <> symbols, in which case the URI will be displayed as the link text.

Markdown
Result
<https://aletheiaware.com> https://aletheiaware.com
<support@aletheiaware.com> support@aletheiaware.com

Block Quotes

Block Quotes are formed by a > symbol, followed by a space, and finally followed by the quote.

Markdown
Result

Code

Code Span

Code Spans are formed by enclosing text in ` symbols.

Markdown
Result
To print "Hello World" in Go use `log.Println("Hello World")`.

To print "Hello World" in Go use log.Println("Hello World").

Indented Code Block

Indented Code Blocks are formed by indenting the code by at least 4 spaces.

Markdown
Result
This is how you can print "Hello World" in Go;

    log.Println("Hello World")

This is how you can print "Hello World" in Go;

log.Println("Hello World")

Fenced Code Blocks

Fenced Code Blocks are formed by surrounding the code with 3 ` or ~ symbols.

Markdown
Result
This is how you can print "Hello World" in Go;
```
log.Println("Hello World")
```

This is how you can print "Hello World" in Go;

log.Println("Hello World")
This is how you can print "Hello World" in Go;
~~~
log.Println("Hello World")
~~~

This is how you can print "Hello World" in Go;

log.Println("Hello World")
